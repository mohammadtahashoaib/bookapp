Our Application

User Registration

Users can create an account by providing necessary details. Email verification is required to activate the account.

Book Search

Users can search for books using keywords or author names. Search results should be relevant and displayed in a user-friendly manner.

Add to Favorites

Users can mark books as favorites to create a personalized reading list. A dedicated section to view and manage favorite books.

Responsive Design

Ensure that the app is responsive and works well on various devices, including smartphones and tablets.

Book Reading

Users can view the contents of the books they select. An intuitive and user-friendly interface for reading books.

Forgot Password

Users can request a password reset in case they forget their password. A secure reset link is sent to the registered email for password recovery.
